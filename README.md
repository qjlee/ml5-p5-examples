# ml5-p5-examples
ml5.js example to use the image classifier via image file upload

```bash
$ git clone https://github.com/qjlee/ml5-p5-examples.git
$ cd ml5-p5-examples
$ npm i -g serve
$ serve
```

open your browser and enjoy!
